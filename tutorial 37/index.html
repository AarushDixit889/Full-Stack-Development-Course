Selectors

Descendant selector This selector refers to inner element.For example-
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<link rel="stylesheet" href="style.css">
<title>Descendant selector</title>
</head>
<body>
<div>
<h1>Change the color of this heading</h1>
</div>
</body>
</html>
 div h1{ color:red; }
Universal selector It selects all the elements
Grouping selector It can select multiple items by ones like h1,h2,h3,h4{code}
